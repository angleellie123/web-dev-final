Scratch Page: Travel + Photography

Built from a blank HTML file (no Bootstrap). This page demonstrates core HTML/CSS requirements: nested lists, image, YouTube embed, on-page anchor, custom background, custom stylesheet, div layout, and a live interactive Tableau embed.

Navigation: Home · Résumé · Web App · Jump to Tableau ↓

1) Trip Planning Checklist (Nested List)

Requirement: an unordered list embedded in an ordered list (or vice versa).

  1. Before the trip
    • Pick dates and book flights/lodging
    • Plan 2–3 “must-do” spots (leave space for spontaneity)
    • Build a simple budget (food + transit + activities)
  2. Pack camera + essentials
    • Camera + extra battery
    • Small tripod
    • SD cards + a backup plan
  3. Capture the story
    • Wide shots to set the scene
    • Medium shots for context
    • Close-ups for detail
